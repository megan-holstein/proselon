# Print Cover

Walk-through for creating print-ready cover PDFs for KDP paperback and hardcover publishing. The paperback cover is created first; the hardcover cover reuses the same elements resized to a new template. This is a reference document — the agent does not perform these steps, but can explain any step or troubleshoot issues on request.

**Resolution note:** All cover assets (ebook cover image, spine text, back-jacket elements) must meet or exceed the dimensions specified in the KDP cover template. Working with undersized assets will produce a low-quality print cover. Ensure source images are at least 300 DPI at the final print dimensions.

## Before You Start

Complete all ebook publishing tasks first:

- Ebook cover finalized
- Vellum EPUB/KDP export complete
- Ebook successfully uploaded to KDP

You will also need:

- The finished ebook cover image
- A photo editing application (Affinity Photo recommended; Photoshop or similar will also work)
- Access to the [KDP cover calculator](https://kdp.amazon.com/en_US/cover-calculator) (template generator for both paperback and hardcover)
- Access to ChatGPT (or another tool) for generating spine and back-jacket copy

## Procedure

### 1. Generate Vellum Paperback

Open the book in Vellum and generate the paperback output. This sets the interior PDF, page count, and trim size that the cover template must match.

### 2. Download the KDP Paperback Cover Template

Go to the [KDP cover calculator](https://kdp.amazon.com/en_US/cover-calculator). Select paperback, then enter the trim size, page count, paper type, and bleed settings from the Vellum paperback. Download the resulting template PDF.

### 3. Set Up the Template in Your Photo Editor

Open the KDP template PDF in Affinity Photo (or your editor of choice). Resize or configure the canvas to match the exact dimensions specified in the template.

### 4. Place the Ebook Cover

Import the finished ebook cover image and position it on the front-cover area of the template, aligning it with the template guides.

### 5. Generate Spine and Back-Jacket Copy

Use ChatGPT (or draft manually) to generate:

- **Spine text** — Book title and author name, sized to fit the spine width from the template.
- **Back-jacket copy** — A short blurb, pull quote, or description suitable for the back cover.

### 6. Place Spine and Back-Jacket Copy

Add the spine text and back-jacket copy to the template, positioning them within the designated spine and back-cover areas. Use the template guide lines to ensure correct alignment and safe margins.

### 7. Export to Print-Ready PDF

Export the completed cover as a PDF, using the print-ready settings required by KDP (PDF/X-1a or equivalent, CMYK color space, fonts embedded, at least 300 DPI).

### 8. Upload to KDP

Upload the print-ready cover PDF to the KDP paperback setup page. Use the KDP previewer to verify alignment, spine placement, and bleed before approving.

## Hardcover

After the paperback is published, repeat the process for hardcover:

### 9. Download the KDP Hardcover Cover Template

Return to the [KDP cover calculator](https://kdp.amazon.com/en_US/cover-calculator). Select hardcover and enter the hardcover specifications (trim size, page count, paper type, bleed). Download the resulting hardcover template PDF.

### 10. Resize Paperback Cover Elements to Hardcover Template

Open the hardcover template in your photo editor. Import the cover elements from the paperback project (front cover image, spine text, back-jacket copy) and resize/reposition them to fit the hardcover template dimensions and guide lines. The hardcover spine will typically be wider than the paperback spine.

### 11. Export and Upload Hardcover Cover

Export to a print-ready PDF using the same settings as the paperback (PDF/X-1a or equivalent, CMYK, fonts embedded, 300+ DPI). Upload to the KDP hardcover setup page and verify with the previewer.
