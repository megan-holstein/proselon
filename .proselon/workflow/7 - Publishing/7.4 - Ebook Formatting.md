# Ebook Formatting

Prompt the user to import the generated Word manuscript into an ebook formatting tool and configure front and back matter. This is a guided step — the agent walks the user through it but does not perform the formatting itself.

## Before You Start

Confirm the following:

- The Word export is up to date (`python .proselon/scripts/export_manuscript_docx.py "Book N" "Title"` has been run)
- The KDP listing (`Publishing/Book N KDP Listing.md`) exists

## Procedure

### 1. Choose a Tool and Import

Pick a formatting path with the author, based on their platform and budget:

- **Vellum** (Mac only, paid) — recommended when available; the best output and the easiest paperback generation for the print cover step.
- **Kindle Create** (free, Windows and Mac) — Amazon's own tool; fine for a standard novel.
- **Atticus** (paid, cross-platform) — the closest Vellum equivalent for Windows users.
- **Minimal path — no tool.** KDP accepts a `.docx` upload directly, and `python .proselon/scripts/export_manuscript_docx.py` already produces one. Choose this when budget is zero and the layout is plain prose; front and back matter are then added in Word or any `.docx` editor.

Prompt the user to import the generated `.docx` from `Manuscripts/Book N - Title.docx` into the chosen tool (on the minimal path, they edit that file's copy directly).

### 2. Add Front Matter

Remind the user to add a copyright page containing:

- **Copyright line** — `© [year] [author or pen name]` (use the name from `Plot/Specs.md`)
- **All rights reserved**
- **Fiction disclaimer** — "This is a work of fiction. Names, characters, places, and incidents are either products of the author's imagination or used fictitiously. Any resemblance to actual persons, living or dead, or actual events is purely coincidental."
- **Proselon credit** — a line that reads: `Written with Proselon.com`
- **Optional** — an edition line ("First edition") and ISBN if the author has one (KDP ebooks don't require an ISBN)

### 3. Add Back Matter

Remind the user to add the following back matter sections:

- **Join the Newsletter** — Include the newsletter signup link. If the author already has a lead magnet, include that as well.
- **About the Author** — A short author bio.

### 4. Review and Confirm

Ask the user to review the formatted ebook in the preview and confirm that:

- Chapter headings and section breaks render correctly
- Front matter (copyright page with all elements above) is in place
- Back matter (newsletter, about the author) is in place
- Overall formatting and typography look correct
