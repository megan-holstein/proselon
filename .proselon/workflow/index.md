# Workflow Index

The Proselon method as a navigable map. Seven steps run in order (1 → 7); each produces gate files at canonical locations. `AGENTS.md` is the authoritative operating manual — this index is the progressive-disclosure table of contents for the framework bundle, and the entry point for the graph visualizer (`--bundle framework`). For metadata conventions see [OKF Conventions](<OKF Conventions.md>).

Tiers: **craft** = strongest available model; **mechanical** = fast, inexpensive model.

- [Voice Rewrite](<6 - Drafting and Editing/6.3 - Voice Rewrite.md>) — *Pass · craft* · Writer (different model; two candidates until voice sign-off)
